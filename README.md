# Edit Sanrio Putra - Developer Portfolio 🚀

A modern, responsive, and highly interactive Single Page Application (SPA) portfolio built for a Mobile Developer & AI Integrator. Designed with premium aesthetics including a Dark Theme, Glassmorphism, and smooth scroll micro-animations.

## 🌟 Features

- **Premium UI/UX**: Dark mode by default with vibrant neon gradient accents and floating blur blobs.
- **Glassmorphism**: Transparent, frosted-glass card effects for projects and contact sections.
- **Dynamic Content**: All data (skills, projects, profile) is separated from the UI and loaded dynamically from `data.js`.
- **Zero Dependencies**: Built with 100% Vanilla HTML5, CSS3, and JavaScript. Extremely lightweight and fast.
- **SVG Logo & Favicon**: Crisp, scalable, and responsive custom-built logo.
- **GitHub Pages Ready**: Can be instantly hosted on GitHub Pages without any build steps.

## 🛠️ Tech Stack

- **HTML5**: Semantic layout and structure.
- **CSS3**: Custom properties (variables), Flexbox/Grid layouts, CSS transitions, and animations.
- **JavaScript (ES6)**: DOM manipulation, Intersection Observer API for scroll animations, and dynamic data rendering.

## 🚀 How to Run Locally

Since this is a vanilla static site, no complex setup is required:
1. Clone this repository.
2. Double-click the `index.html` file to open it in your browser.
3. Alternatively, use an extension like **Live Server** in VS Code for hot-reloading.

## 📝 How to Update Content

You don't need to touch the HTML to update your portfolio! 
Simply open `data.js` and modify the `portfolioData` object. You can easily:
- Update your bio/summary.
- Add or remove technical skills.
- Add new projects or update existing App Store links and screenshot URLs.

## 🌐 Deployment (GitHub Pages)

1. Make sure your repository is named exactly `edita7x.github.io` for a root-level domain.
2. Push the code to the `master` or `main` branch.
3. Go to your repository **Settings** > **Pages**.
4. Set the source to deploy from your main branch.
5. Your portfolio will be live at `https://edita7x.github.io` in minutes!
